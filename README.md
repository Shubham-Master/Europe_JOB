## 🚀 Setup

```bash
git clone https://github.com/Shubham-Master/Europe_JOB.git
cd Europe_JOB
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
# Add your API keys to .env
```

## ⚙️ Usage

```bash
# Parse your CV
```
